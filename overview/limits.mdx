## Credit System

<Note>
  Free tier accounts require credits for certain operations. Operations will fail if insufficient credits are available.
</Note>

<AccordionGroup>
  <Accordion title="Credit Requirements">
    - Minimum credit requirement varies by operation
    - Credit count is checked before each operation
    - Credits are deducted upon successful completion
  </Accordion>
  
  <Accordion title="Credit Management">
    - Monitor your credit balance in the dashboard
    - Set up alerts for low credit warnings
    - Auto-reload options available for Pro tier
  </Accordion>
</AccordionGroup>

## Error Responses

    ```json
    {
      "error": "TooManyRequests",
      "message": "Rate limit exceeded",
      "details": {
        "rate_limit": "<limit>",
        "api_name": "<endpoint>"
      }
    }
    ```

    ```json
    {
      "error": "Unauthorized",
      "message": "Insufficient credits. Please upgrade your account.",
      "details": {
        "required_credits": "<required>",
        "current_credits": "<available>"
      }
    }
    ```

    ```json
    {
      "error": "BadRequest",
      "message": "File size exceeds maximum limit of 1 GB"
    }
    ```


<Warning>
  Repeated rate limit violations may result in temporary account suspension.
</Warning>
