</Note>

## Cloud Infrastructure Options

<AccordionGroup>
  <Accordion title="Amazon Web Services (AWS)">
    Deploy Mixpeek in your AWS VPC with full integration for IAM roles, S3 storage, and marketplace billing.
  </Accordion>

  <Accordion title="Google Cloud Platform (GCP)">
    Run Mixpeek in your GCP environment with service account integration and Vertex AI optimization.
  </Accordion>
</AccordionGroup>

## BYO API Keys

For select managed models, you can use your existing API keys:

- **Google Vertex AI**: Use your GCP credentials for Vertex models
- **Hugging Face**: Connect your Inference Endpoints


<Note>For custom models, see the <a href="/overview/models/available-models#3-custom-model-integration">Custom Model Integration section</a></Note>

All API keys are:
- Securely tied to your organization
- Encrypted at rest
- Isolated per namespace

Learn about our <a href="https://mixpeek.com/security">security practices and compliance standards</a>
