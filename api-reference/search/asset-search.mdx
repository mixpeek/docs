# Asset Search API

Search for entire assets based on their content and metadata using natural language queries.

## Endpoint

```bash
POST /v1/search/assets
```

## Request Body

```json
{
  "query": "sales presentation about cloud computing",
  "namespace": "marketing",
  "collection": "presentations",
  "filters": {
    "file_type": "pptx",
    "date_created": {
      "gte": "2023-01-01"
    }
  },
  "limit": 10,
  "offset": 0
}
```

### Parameters

| Parameter    | Type    | Required | Description                                       |
| ------------ | ------- | -------- | ------------------------------------------------- |
| `query`      | string  | Yes      | Natural language search query                     |
| `namespace`  | string  | Yes      | Namespace to search within                        |
| `collection` | string  | No       | Optional collection to narrow search scope        |
| `filters`    | object  | No       | Additional filters to apply to the search         |
| `limit`      | integer | No       | Maximum number of results (default: 10, max: 100) |
| `offset`     | integer | No       | Number of results to skip for pagination          |

### Filter Options

| Filter          | Type   | Description                                       |
| --------------- | ------ | ------------------------------------------------- |
| `file_type`     | string | Filter by file extension                          |
| `date_created`  | object | Filter by creation date with `gte` and `lte`      |
| `date_modified` | object | Filter by modification date with `gte` and `lte`  |
| `size`          | object | Filter by file size in bytes with `gte` and `lte` |
| `metadata`      | object | Filter by custom metadata fields                  |

## Response

```json
{
  "results": [
    {
      "id": "asset_789ghi",
      "name": "Cloud_Computing_Overview_2023.pptx",
      "file_type": "pptx",
      "size": 2048576,
      "score": 0.89,
      "metadata": {
        "author": "Jane Smith",
        "department": "Marketing",
        "tags": ["cloud", "presentation"]
      },
      "date_created": "2023-03-15T14:30:00Z",
      "date_modified": "2023-03-15T14:30:00Z",
      "feature_count": 45
    }
  ],
  "total": 23,
  "offset": 0,
  "limit": 10
}
```

### Response Fields

| Field           | Type    | Description                  |
| --------------- | ------- | ---------------------------- |
| `id`            | string  | Unique asset identifier      |
| `name`          | string  | Original filename            |
| `file_type`     | string  | File extension               |
| `size`          | integer | File size in bytes           |
| `score`         | number  | Relevance score (0-1)        |
| `metadata`      | object  | Custom metadata              |
| `date_created`  | string  | Creation timestamp           |
| `date_modified` | string  | Last modified timestamp      |
| `feature_count` | integer | Number of extracted features |

## Example Requests

### Basic Search

```bash
curl -X POST https://api.mixpeek.com/v1/search/assets \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "sales presentation",
    "namespace": "marketing"
  }'
```

### Filtered Search

```bash
curl -X POST https://api.mixpeek.com/v1/search/assets \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "sales presentation",
    "namespace": "marketing",
    "filters": {
      "file_type": "pptx",
      "date_created": {
        "gte": "2023-01-01"
      }
    }
  }'
```

## Error Responses

| Status Code | Description                |
| ----------- | -------------------------- |
| `400`       | Invalid request parameters |
| `401`       | Invalid or missing API key |
| `403`       | Insufficient permissions   |
| `404`       | Namespace not found        |
| `429`       | Rate limit exceeded        |
