---
title: "Get One"
description: "Retrieve a file from the index by its' unique file id."
api: "GET /v1/file/{file_id}"
---

## Request

<ParamField path="file_id" type="string" required>
  File ID of the file you'd like retrieved.
</ParamField>

<RequestExample>

```python Python
import requests

url = "https://api.mixpeek.com/v1/file?file_id=63a47050660c021b5027166e"

headers = {'Authorization': 'API_KEY'}

requests.request("GET", url, headers=headers)
```

</RequestExample>
