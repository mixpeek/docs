---
title: "Index One"
description: "Securely send your file to the Mixpeek server, where the content is extracted, indexed and then searchable. Files are never stored on our servers."
api: "POST /v1/file/index"
---

## Request

<ParamField body="file" type="file" required>
  Provide your file.
</ParamField>

<ParamField query="save" type="boolean" optional>
  If `true`, it will save it in Mixpeek's dedicated S3 bucket. Your file will be
  stored within a unique folder to your api-key tenant. This will override any
  `static_file_url` provided and will include it in the response.
</ParamField>

<ParamField body="description" type="string" optional>
  Description of the file.
</ParamField>

<ParamField body="user_id" type="string" optional>
  Filter used to run isolated searches on individual user's files.
</ParamField>

<ParamField body="tags" type="string" optional>
  List of tags which can be used to influence search results.
</ParamField>

<ParamField body="static_file_url" type="string" optional>
  Used as a reference to associate with your file.
</ParamField>

## Response

<ResponseField name="file_id" type="string" required>
  Your indexed files' unique identifier.
</ResponseField>

<RequestExample>

```python Python
from mixpeek import Mixpeek

mix = Mixpeek(api_key="API_KEY")

# index one local pdf document without any extra metadata
mix.index("/user/desktop/file.pdf")

# with extra metadata
mix.index(
  "/user/desktop/file.pdf",
  # everything below is optional
  description="This file contains a cat.",
  user_id="123",
  tags="document, legal",
  static_file_url="cdn.host.com/file.pdf",
  save=True
)
```

```shell Shell
curl --location --request POST 'https://api.mixpeek.com/v1/file/index' \
--header 'Authorization: API_KEY' \
--form 'file=@"/path/to/file.pdf"' \
--form 'user_id="john_smith_123"' \
--form 'tags="audio, file, today"' \
--form 'static_file_url="s3://file.pdf"'
}'
```

```javascript Javascript
var myHeaders = new Headers();
myHeaders.append("Authorization", "API_KEY");

var formdata = new FormData();
formdata.append("file", fileInput.files[0], "file.pdf");
formdata.append("user_id", "john_smith_123");
formdata.append("tags", "document, file, today");
formdata.append("static_file_url", "s3://file.pdf");

var requestOptions = {
  method: "POST",
  headers: myHeaders,
  body: formdata,
  redirect: "follow",
};

fetch("https://api.mixpeek.com/v1/file/index", requestOptions)
  .then((response) => response.text())
  .then((result) => console.log(result))
  .catch((error) => console.log("error", error));
```
