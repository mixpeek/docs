---
title: "Search"
description: "This searches your files for the text provided."
api: "GET /v1/search?q={query}"
---

### Query Parameters

<ParamField query="q" required type="string">
  Your search query
</ParamField>

### Request Body

<ParamField body="user_id" type="string">
  Filter, used to run isolated queries against individual user's files.
</ParamField>

<ParamField body="include_context" type="boolean" default="false">
  Indicate where the keyword exists within the body of text from the file.
</ParamField>

<ParamField body="semantics" type="boolean" default="false">
    Include the semantically similar terms in your results.
</ParamField>

<ParamField body="clauses" type="array">
  Array of objects for clauses to the search engine. Each object must contain
  keys:            operator (OR, AND, AND NOT) and query.
</ParamField>

<RequestExample>

```python Python
import requests

url = "https://api.mixpeek.com/search?q=SEARCH_QUERY"

headers = {
  'Authorization': 'API_KEY'
}

response = requests.request("GET", url, headers=headers)

print(response.text)
```

```shell Shell
curl --location --request GET 'https://api.mixpeek.com/search?q=SEARCH_QUERY' \
--header 'Authorization: API_KEY'
```

```javascript JavaScript
var myHeaders = new Headers();
myHeaders.append("Authorization", "API_KEY");

var requestOptions = {
  method:          "GET",
  headers:         myHeaders,
  redirect:        "follow",
};

fetch("https://api.mixpeek.com/search?q=", requestOptions)
  .then((response) => response.text())
  .then((result) => console.log(result))
  .catch((error) => console.log("error", error));
```

```ruby Ruby
require "uri"
require "net/http"

url = URI("https://api.mixpeek.com/search?q=SEARCH_QUERY")

https = Net::HTTP.new(url.host, url.port)
https.use_ssl = true

request = Net::HTTP::Get.new(url)
request["Authorization"] = "API_KEY"

response = https.request(request)
puts response.read_body
```
