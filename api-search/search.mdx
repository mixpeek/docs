---
title: "Search"
description: "This searches your files for the text provided."
api: "GET /v1/search"
---

### Request

<ParamField query="q" type="string" required>
    Your search query.
</ParamField>

<ParamField body="user_id" type="string">
    **Optional** Filter used to run isolated queries against individual user's files.
</ParamField>

<ParamField body="include_context" type="boolean" default="False">
    **Optional** Indicate where the keyword exists within the body of text from the file.
</ParamField>

<ParamField body="semantics" type="boolean" default="False">
    **Optional** Include the semantically similar terms in your results.
</ParamField>

## Response

<ResponseField name="file_id" type="string" required>
    Your uploaded files' unique identifier.
</ResponseField>

<RequestExample>

```python Python
import requests

url = "https://api.mixpeek.com/v1/search?q=prescription"
payload = {
    "user_id": "john_smith_123",
    "include_context": False,
    "semantics": False
}
headers = {'Authorization': 'API_KEY'}

requests.request("GET", url, data=payload, headers=headers)
```

```shell Shell
curl --location --request GET 'https://api.mixpeek.com/v1/search?q=prescription' \
--header 'Authorization: API_KEY' \
--header 'Content-Type: application/json' \
--data-raw '{
    "user_id": "john_smith_123",
    "include_context": false,
    "semantics": false
}'
```

```javascript Javascript
var myHeaders = new Headers();
myHeaders.append("Authorization", "API_KEY");
myHeaders.append("Content-Type", "application/json");

var raw = JSON.stringify({
  "user_id": "john_smith_123",
  "include_context": false,
  "semantics": false
});

var requestOptions = {
  method: 'GET',
  headers: myHeaders,
  body: raw,
  redirect: 'follow'
};

fetch("https://api.mixpeek.com/v1/search?q=prescription", requestOptions)
  .then(response => response.text())
  .then(result => console.log(result))
  .catch(error => console.log('error', error));
```
