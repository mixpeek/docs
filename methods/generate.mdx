---
title: "Generate"
description: "Generates content based on provided input for text, image, audio, or video."
api: "POST /generate/{modality}"
---

## Request

<ParamField body="model" type="string" required>
  The model to use for content generation
</ParamField>

<ParamField body="content" type="string" required>
  The prompt or input to generate content from
</ParamField>

<ParamField body="parameters" type="json" optional>
  Additional parameters specific to the model and modality
</ParamField>

<RequestExample>

```shell Shell
curl --location 'https://api.mixpeek.com/generate/{modality}' \
--header 'Authorization: Bearer API_KEY' \
--header 'Content-Type: application/json' \
--data '{
    "model": "openai/dall-e",
    "content": "A futuristic cityscape",
    "parameters": {
        "size": "1024x1024"
    }
}'
```
</RequestExample>
