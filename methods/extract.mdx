---
title: "Extract"
description: "Pulls information out of the text, image, audio or video."
api: "POST /extract/{modality}"
---

## Request

<ParamField body="model" type="string" required>
  The model to use for extraction
</ParamField>

<ParamField body="url" type="string" optional>
Remote url of the file to extract
</ParamField>

<ParamField body="content" type="boolean" optional>
  The direct textual or base64-encoded content of the document for extraction.
</ParamField>



<RequestExample>

```shell Shell
curl --location 'https://api.mixpeek.com/extract/{modality}' \
--header 'Authorization: Bearer API_KEY' \
--header 'Content-Type: application/json' \
--data '{
    "model": "dslim/bert-base-NER",
    "url": "https://example.com/document.pdf",
}'
```
</RequestExample>
