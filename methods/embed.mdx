---
title: "Embed"
description: "Generates embeddings for text, image, audio, or video data to create machine-readable representations."
api: "POST /embed/{modality}"
---

## Request

<ParamField body="model" type="string" required>
  The model to use for embedding
</ParamField>

<ParamField body="url" type="string" optional>
Remote url of the file to be embedded
</ParamField>

<ParamField body="content" type="boolean" optional>
  The direct textual or base64-encoded content of the item for embedding.
</ParamField>

<RequestExample>

```shell Shell
curl --location 'https://api.mixpeek.com/embed/{modality}' \
--header 'Authorization: Bearer API_KEY' \
--header 'Content-Type: application/json' \
--data '{
    "model": "openai/clip",
    "url": "https://example.com/image.jpg",
}'
```
</RequestExample>
