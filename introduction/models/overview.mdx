The lists below show all the models currently supported in the SDK for various tasks. For embedding models, use the dimensions to create your DB index.

<Note>
  We add models regularly, so if we're missing any
  <a href="https://mixpeek.com/contact">reach out</a>.
</Note>

### Embedding Models

Embedding models convert data into numerical vectors, enabling efficient similarity searches and machine learning tasks.

| Name                           | Modality    | Dimensions | Description                                     | URL                                                           |
| ------------------------------ | ----------- | ---------- | ----------------------------------------------- | ------------------------------------------------------------- |
| `openai-clip-vit-base-patch32` | Text, Image | 512        | A versatile model for text and image embeddings | [OpenAI Clip](https://github.com/openai/CLIP)                 |
| `vuse-generic-v1`              | Text, Video | 768        | Specialized for text and video embeddings       | [Mixpeek](https://mixpeek.com/models/mixpeek/vuse-generic-v1) |

### Read Models

Read models extract and interpret information from various data types, providing structured output.

| Name                  | Modality | Description                                              | URL                                                               |
| --------------------- | -------- | -------------------------------------------------------- | ----------------------------------------------------------------- |
| `video-descriptor-v1` | Video    | Extracts key information and metadata from video content | [Mixpeek](https://mixpeek.com/models/mixpeek/video-descriptor-v1) |

### Describe Models

Describe models generate human-readable descriptions or summaries of input data.

| Name                  | Modality | Description                                      | URL                                                               |
| --------------------- | -------- | ------------------------------------------------ | ----------------------------------------------------------------- |
| `video-descriptor-v1` | Video    | Generates detailed descriptions of video content | [Mixpeek](https://mixpeek.com/models/mixpeek/video-descriptor-v1) |

### Transcribe Models

Transcribe models convert spoken language in audio or video into written text.

| Name          | Modality | Description                                | URL                                                       |
| ------------- | -------- | ------------------------------------------ | --------------------------------------------------------- |
| `polyglot-v1` | Audio    | Transcribes speech from multiple languages | [Mixpeek](https://mixpeek.com/models/mixpeek/polyglot-v1) |
