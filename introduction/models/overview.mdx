The list below are all the embedding models currently supported in the SDK. For embed, use the dimensions to create your DB index.

<Note>
  We add models regularly, so if we're missing any
  <a href="https://mixpeek.com/contact">reach out</a>.
</Note>

### Embedding Models

| Name                             | Modality    | Dimensions | URL                                                           |
| -------------------------------- | ----------- | ---------- | ------------------------------------------------------------- |
| `all-MiniLM-L6-v2`               | Text        | 384        | [Sentence Transformers](https://www.sbert.net/)               |
| `nomic-embed-text-v1`            | Text        | 768        | [Nomic AI](https://nomic.ai/)                                 |
| `jina-embeddings-v2-base-en`     | Text        | 768        | [Jina AI](https://jina.ai/)                                   |
| `bert-base-multilingual-uncased` | Text        | 768        | [Google BERT](https://github.com/google-research/bert)        |
| `mxbai-embed-large-v1`           | Text        | 1024       | [MixedBread AI](https://mixedbread.ai/)                       |
| `clip-vit-base-patch32`          | Text, Image | 512        | [OpenAI Clip](https://github.com/openai/CLIP)                 |
| `vuse-generic-v1`                | Text, Video | 768        | [Mixpeek](https://mixpeek.com/models/mixpeek/vuse-generic-v1) |
