We have created a few libraries to help you integrate with the Mixpeek API. We'll continue to add support for more libraries. Email [info@mixpeek.com](mailto:info@mixpeek.com) for specific library support.

### Supported Libraries

|                                                                  |                                                      |
| ---------------------------------------------------------------- | ---------------------------------------------------- |
| [mixpeek-python](https://github.com/mixpeek/mixpeek-python)               | Python library for calling Mixpeek API on your backend |
| Node.js | Soon to be added                                  |
| Java                                                             | Soon to be added                                     |
| Go                                                               | Soon to be added                                     |
