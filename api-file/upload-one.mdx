---
title: "Upload"
description: "Securely send your file(s) to our server where its' content is extracted, indexed and searchable via /search. Files are never stored on our servers."
api: "POST /v1/file/upload"
---

## Request

<ParamField body="file_url" type="array" required>
    Static hosted url of the file(s). Can provide multiple file urls and they will be uploaded in a background process.
</ParamField>

<ParamField body="semantics" type="boolean" default="False">
    **Optional** Set to `True` if you wish to perform semantic search, learn more about semantic search <a href="#">here</a>.
</ParamField>

<ParamField body="user_id" type="string">
    **Optional** Filter used to run isolated searches on individual user's files.
</ParamField>

<ParamField body="tags" type="array">
    **Optional** List of *tags* which can be used to influence search results.
</ParamField>

## Response

<ResponseField name="file_id" type="string" required>
    Your uploaded files' unique identifier.
</ResponseField>

<RequestExample>

```python Python
import requests

url = "https://api.mixpeek.com/v1/file/upload"
payload = {
    "file_url": ["https://mixpeek-demo.s3.us-east-2.amazonaws.com/prescription.pdf"],
    "semantics": True,
    "user_id": "john_smith_123",
    "tags": ["veterinarian", "medical", "dog"]
}
headers = {'Authorization': 'API_KEY'}

requests.request("POST", url, data=payload, headers=headers)
```

``` javascript Javascript
var myHeaders = new Headers();
myHeaders.append("Authorization", "API_KEY");
myHeaders.append("Content-Type", "application/json");

var raw = {
    "file_url": "https://mixpeek-demo.s3.us-east-2.amazonaws.com/prescription.pdf",
    "semantics": True,
    "user_id": "john_smith_123",
    "tags": ["veterinarian", "medical", "dog"]
};

var requestOptions = {
  method: 'POST',
  headers: myHeaders,
  body: raw,
  redirect: 'follow'
};

fetch("https://api.mixpeek.com/v1/file/upload", requestOptions)
  .then(response => response.text())
  .then(result => console.log(result))
  .catch(error => console.log('error', error));
```
