---
title: "Upload One"
description: "Securely send your file to our server where its' content is extracted, indexed and searchable via /search."
api: "POST /v1/file/upload-one"
---

## Request Body

<ParamField body="file" type="file" required>
    Provide your file in the `file` form-data part.
</ParamField>

<ParamField body="semantics" type="boolean" default="false">
    Required if you want to perform semantic search, learn more <a href="#">here</a>.
</ParamField>

<ParamField body="user_id" type="string">
    Used for you to create user access controls within your application.
</ParamField>

<ParamField body="static_file_path" type="array">
    Static hosted url of the file, for reference in the response.
</ParamField>

<ParamField body="tags" type="array">
    List of *tags* which can be used to influence search results.
</ParamField>

## Response Body

<ParamField body="file_id" type="string">
    Your uploaded files' unique identifier.
</ParamField>

<RequestExample>

```python Python
import requests

url = "https://api.mixpeek.com/v1/file/upload-one"
payload = {
    "user_id": "john_smith_123",
    "include_context": False,
    "semantics": False
}
headers = {'Authorization': 'API_KEY'}
files = [('file', ('document.pdf', open(
    '/local/path/document.pdf', 'rb'), 'application/pdf'))]

requests.request("POST", url, data=payload, headers=headers, files=files)
```

</RequestExample>

<ResponseExample>

```json Response
{"file_id":"123"}
```

</ResponseExample>
