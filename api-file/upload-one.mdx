---
title: "Upload One"
description: "Securely send your file to our server where its' content is extracted, indexed and searchable via /search."
api: "POST /file/upload-one"
---

<ParamField body="file" type="file" required>
    Provide your file in the `file` form-data part.
</ParamField>

<ParamField body="semantics" type="boolean">
    Required if you want to perform semantic search, learn more <a href="#">here</a>.
</ParamField>

<ParamField body="user_id" type="string">
    Used for you to create user access controls within your application.
</ParamField>

<ParamField body="static_file_path" type="array">
    Static hosted url of the file, for reference in the response.
</ParamField>

<ParamField body="tags" type="array">
    List of *tags* which can be used to influence search results.
</ParamField>


## Note:
- Your files are promptly deleted after its' text is extracted. See the <a href="https://mixpeek.com/security">Security Page</a> to learn more.
- Navigate to the <a href="#">file types</a> page to see which file types we currently support.
