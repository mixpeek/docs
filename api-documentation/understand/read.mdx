---
title: "Read"
description: "Extracts and reads text from video content based on a given URL and time range."
api: "POST /read/url"
---

<Warning>This feature is in beta and may change without notice.</Warning>

## Request

<ParamField body="url" type="string" required>
  The URL of the video file to extract text from.
</ParamField>

<ParamField body="model_id" type="string" required>
  The ID of the model to use for text extraction. Default is
  `video-descriptor-v1`.
</ParamField>

## Response

The API returns a JSON object containing the extracted text from the specified video segment.

<ResponseExample>

```json JSON
{
  "text": "Welcome to our tutorial on machine learning basics. In this video, we'll cover the fundamental concepts of supervised and unsupervised learning."
}
```

</ResponseExample>
