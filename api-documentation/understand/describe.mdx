---
title: "Describe"
description: "Generates descriptions for video content based on a given prompt and URL."
api: "POST /describe/url"
---

<Warning>This feature is in beta and may change without notice.</Warning>

## Request

<ParamField body="url" type="string" required>
  The URL of the video file to be described.
</ParamField>

<ParamField body="prompt" type="string" required>
  The description or prompt of what to look for in the video.
</ParamField>

<ParamField body="model_id" type="string" required>
  The ID of the model to use for video description. Default is
  "video-descriptor-v1".
</ParamField>

## Response

The API returns a JSON object containing the structured description of the video based on the given prompt.

<ResponseExample>

```json JSON
{
  "description": "The video shows a cartoon animation of a small white rabbit hopping through a prehistoric landscape with large dinosaurs in the background."
}
```

</ResponseExample>

## Notes

- The API uses the `video-descriptor-v1` model for video description.
- Ensure that the provided URL is publicly accessible.
