---
title: "Transcribe"
description: "Generates text transcriptions from audio or video data."
api: "POST /transcribe/url"
---

## Request

<ParamField body="url" type="string" required>
  The URL of the audio or video file to be transcribed.
</ParamField>

## Response

<ResponseField name="transcription" type="string">
  The generated text transcription of the audio or video content.
</ResponseField>

<ResponseExample>

```json Response
{
  "transcription": "or something? Ah! Ah!"
}
```

</ResponseExample>
