---
title: "Embed"
description: "Generates embeddings for text, image, audio, or video data to create machine-readable representations."
api: "POST /embed"
---

## Request

<ParamField body="model_id" type="string" required>
  The model_id to use for embedding,
</ParamField>

<ParamField body="input" type="string" required>
  The input source for the file to be embedded, e.g., a URL or direct content.

  <Warning>
  Max of 5mb per request can be sent (via url or base64), consider using the [File Tools API](/api-documentation/tools/video) to pre-process larger inputs
  </Warning>
</ParamField>

<ParamField body="modality" type="string" required>
  The type of data to be embedded (e.g., text, image, audio, video).
</ParamField>

<ParamField body="input_type" type="string" required>
  The type of input provided (e.g., "url" for remote URLs, "text" for text or
  "base64" for base64-encoded content).
</ParamField>
