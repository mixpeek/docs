Mixpeek is designed to seamlessly integrate with various datastores and file storage systems, providing a robust pipeline to process and enrich your data. Integrations enable you to connect your existing systems with Mixpeek to extract, generate, and embed rich data across multiple modalities.

## Database Integrations

Mixpeek supports a wide range of database integrations for storing embeddings and metadata:

- MongoDB
- PostgreSQL
- Pinecone
- Weaviate
- Qdrant
- Redis

## File Storage Integrations

We offer integrations with various file storage systems:

- AWS S3
- Google Cloud Storage
- Azure Blob Storage

## File Streaming Integrations

For real-time file streaming, Mixpeek supports:

- RTSP (Real-Time Streaming Protocol)
- WebRTC (Web Real-Time Communication)
- FTP (File Transfer Protocol)

These integrations allow you to connect your existing data infrastructure with Mixpeek, enabling powerful data processing and enrichment capabilities across different modalities and storage systems.
