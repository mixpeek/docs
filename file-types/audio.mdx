### Upload Example

<CodeGroup>

```python upload.py
import requests

url = "https://api.mixpeek.com/upload"

files=[
  ('file',('audio.m4a',open('FILE_PATH','rb'),'application/octet-stream'))
]
headers = {
  'Authorization': 'API_KEY'
}

response = requests.request("POST", url, headers=headers, files=files)

print(response.text)
```

### Search Example

<CodeGroup>

```python search.py
import requests

url = "https://api.mixpeek.com/search?q=application"

headers = {
  'Authorization': API_KEY'
}

response = requests.request("GET", url, headers=headers)

print(response.text)
```
